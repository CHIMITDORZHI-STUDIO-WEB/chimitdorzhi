FAQ

Частые вопросы

На чём пишете ботов?
Python с библиотекой aiogram — это стандарт для production-ботов. Для маленьких ботов или прототипов могу использовать готовые конструкторы (BotHelp, SaleBot), если так дешевле под задачу.
Где будет жить бот?
На вашем VPS (от 200 ₽/мес) или на моём сервере (включено в первые 3 месяца). После — арендуете сами или продлеваете у меня. Доступы и код передаются полностью.
Можно ли подключить AI?
Да. GPT-4 / Claude через API для ответов на вопросы клиентов, классификации заявок, генерации описаний. Считаем расход на токены отдельно — обычно копейки за разговор.
Бот будет работать в групповом чате?
Да, если так нужно. Telegram-боты умеют работать в группах: модерация, опросы, ответы на упоминания. Это отдельный сценарий, проектируем под вашу задачу.
Что с приёмом денег?
Telegram Payments через ЮKassa или Tinkoff — это законный приём с чеками по 54-ФЗ. Подключение занимает 1–3 дня, нужен ИП или ООО. Помогаю с настройкой эквайринга.
