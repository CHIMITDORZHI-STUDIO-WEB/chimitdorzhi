FAQ

Частые вопросы

Работает ли PWA на iPhone?
Да, начиная с iOS 16.4 — с push-уведомлениями. До этого PWA на iOS работали, но без пушей. Сейчас Safari поддерживает большинство фич, кроме нескольких аппаратных API.
Чем PWA хуже нативного приложения?
Нет доступа к части системных API (NFC, Bluetooth LE с ограничениями, контакты). Производительность чуть ниже на тяжёлой графике. Для большинства бизнес-приложений разницы пользователь не заметит.
Можно ли потом опубликовать PWA в магазинах?
Да, через TWA (Trusted Web Activity) для Google Play или PWABuilder для Microsoft Store. Apple App Store напрямую не принимает PWA — нужна обёртка, и они часто отклоняют такое.
Можно ли превратить сайт в PWA?
Если сайт нормальный — да, в большинстве случаев это 2–3 недели работы: manifest, Service Worker, аудит. Если сайт на старом фреймворке без сборки — может потребоваться переписать клиентскую часть.
