Десктопные приложения

Electron и Tauri — приложения, которые ставятся в Windows, macOS и Linux. Веб-стек снаружи, нативные возможности внутри.

Простыми словами. Это веб-приложение, упакованное как обычная программа: пользователь ставит установщик, иконка появляется в меню «Пуск» или в Dock. Внутри — HTML/JS, снаружи — нативное окно с доступом к файловой системе, принтеру и железу.

FAQ

Частые вопросы

Electron или Tauri?
Electron — больше готовых библиотек, проще нанять разработчиков, тяжелее приложение (от 80 МБ). Tauri — приложение в 5–10 раз меньше и быстрее, но стек на Rust сложнее найти. Для большинства задач Electron — рабочая лошадка.
Нужна ли подпись кода?
Для Windows — желательно (иначе SmartScreen ругается), сертификат от 5 000 ₽/год. Для macOS — обязательно для распространения вне Mac App Store, требует Apple Developer (99$/год) и нотаризацию.
Как с автообновлением?
Реализуется через electron-updater / tauri-updater + ваш сервер обновлений (или GitHub Releases). Пользователь получает уведомление и обновляется в один клик.
Можно ли перенести существующее веб-приложение?
Если фронт на React / Vue — можно за 1–2 недели обернуть. Если приложение использует серверные фичи, которые не работают офлайн — нужна доработка.
