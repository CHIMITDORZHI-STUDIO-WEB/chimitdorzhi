FAQ

Частые вопросы

Что с Manifest V3?
Google окончательно перевёл всё на Manifest V3 в 2024. Старые V2-расширения не принимаются. Все новые разработки делаю сразу на V3 — нет смысла писать устаревшее.
Сколько модерируется в Web Store?
Первая публикация — 1–3 дня обычно, иногда до недели. Главные причины отказа — широкие permissions без обоснования или сбор данных без политики приватности. Помогаю с описанием и privacy policy.
Можно ли продавать подписки?
Да. Через Chrome Web Store Payments (для зарубежных) или через свой биллинг (для России — ЮKassa, Tinkoff). Внутри расширения проверяется статус подписки через ваш API.
А Edge и Firefox?
Расширения для Edge почти 1-в-1 как для Chrome — один код, разная публикация. Firefox имеет свои особенности (некоторые API отличаются), но базовое расширение легко портируется.
