FAQ

Частые вопросы

Flutter или React Native — что лучше?
Flutter — лучше графика и анимации, единый рендеринг, быстрее старт. React Native — ближе к нативу, проще нанять разработчиков, переиспользует React-знания из веба. По задаче решаем.
Будут ли тормоза?
Для бизнес-приложений, e-commerce, кабинетов, мессенджеров — никаких заметных тормозов. Для тяжёлой 3D-графики, AR и игр — лучше натив или Unity.
Можно ли потом перейти на натив?
Можно, но это переписывание. Поэтому выбор стека — стратегическое решение в начале проекта. Помогаю с этим выбором в брифе, считаем бизнес-кейс.
Что с обновлениями без релиза в магазин?
Через CodePush (React Native) и аналоги для Flutter — JS-логика обновляется без релиза. Apple это терпит при определённых условиях, Google — спокойно.
