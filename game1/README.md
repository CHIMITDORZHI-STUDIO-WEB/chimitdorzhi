# Толкователь — игра в слова

Веб-игра про **доказательство родства слов**. Под разными алфавитами и звуками
прячется один корень — игрок собирает родство обратно. Игра не выдаёт финальную
этимологию, она будит здоровое любопытство к словам.

**Идея:** Ольга Раднаева. **Исполнитель:** Chimitdorzhi Studio.
Чистый HTML + CSS + JavaScript, без сборщиков. Открывается двойным кликом по `index.html`.

Живёт по адресу: **https://chimitdorzhi.tech/game1/**

## Как устроено (трафарет)

Механика — одна. Новые уровни добавляются **только данными**, без правки кода:

```
game1/
  index.html            оболочка
  css/styles.css        оформление
  js/data.js            ★ ДАННЫЕ: языки, звания, уровни (корни и слова)
  js/engine.js          движок (механика, разбор, проверка) — НЕ редактировать
  js/storage.js         сохранение прогресса — НЕ редактировать
  js/app.js             связывание и навигация — НЕ редактировать
  manifest.webmanifest  установка как приложение
  service-worker.js     работа офлайн
  icons/                иконки приложения
```

Жёсткое правило: весь контент живёт в `js/data.js`. Остальные `.js` не трогаем.

## Как добавить новый уровень

Откройте `js/data.js` и добавьте объект в массив `LEVELS`:

```js
{
  id: 'mur',                  // короткий уникальный код уровня (латиницей)
  root: 'МУР',                // корень, который доказываем
  note: "Вступление. Можно выделить <span class='hl'>ключевой факт</span>.",
  task: 'Что делать игроку: открыть разбор и поставить вердикт по каждому слову.',
  words: [
    {
      lang: 'rus',            // язык: rus | bur | grc | lat | tur | eng
      w: 'МУРАВА',            // слово как пишется
      tr: 'мурава',           // транскрипция (можно оставить пустой '')
      mean: 'рус. — молодая трава',   // значение и перевод
      transform: 'М<keep>У</keep>РАВА <ar>→</ar> У держит «у» <ar>→</ar> <keep>МУР</keep>',
      related: true           // true — родственно корню; false — только созвучие
    }
    // ...ещё слова
  ]
}
```

### Разметка поля `transform` (подсказка-разбор)

В строке `transform` тремя метками показывается превращение букв к корню:

- `<keep>…</keep>` — буква/звук, **сохранившие** связь с корнем (подсветится зелёным);
- `<drop>…</drop>` — где связь **рвётся** или только созвучие (зачёркнётся красным);
- `<ar>→</ar>` — **шаг** превращения (стрелка акцентным цветом).

Пример: `Τ<drop>Υ</drop>ΡΟΣ <ar>→</ar> «тирос» <ar>→</ar> иной корень, созвучие`.

### Новый язык

Если нужного языка нет — добавьте его в карту `LANGS` в начале `data.js`:

```js
const LANGS = {
  // ...
  san: { label: 'санскр', color: '#7a4a5a' }
};
```

### Звания и очки

- За каждый верный вердикт игрок получает **+10 очков** (`POINTS_PER_CORRECT`).
- Уровень засчитывается, когда **все** вердикты верны; следующий уровень открывается после этого.
- Пороги званий — массив `RANKS` в `data.js`
  (Ученик · Чтец словарей · Толкователь · Немой полиглот).

### Заготовки

В `LEVELS` уже лежат две заготовки (`id: 'todo-2'`, `'todo-3'`) с пустым `words`.
Пока массив `words` пуст — уровень показывается как «в подготовке».
Заполните `root`, `note`, `task` и `words` — и уровень оживёт.
