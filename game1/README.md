# Толкователь — игра в слова

Веб-игра на основе алфавита-ключа. Идея: **Ольга Раднаева**. Исполнитель: Chimitdorzhi Studio.
Чистый HTML + CSS + JavaScript, без сборщиков. Открывается двойным кликом по `index.html`.

Живёт по адресу: **https://chimitdorzhi.tech/game1/**

## Как устроено (трафарет)

Механика — одна, новые уровни добавляются **только данными**, без правки кода:

```
game1/
  index.html         — оболочка
  css/styles.css     — оформление
  js/data.js         — ★ ДАННЫЕ: корни, слова, словарь, очки, звания
  js/engine.js       — движок (механика, очки) — НЕ редактировать
  js/storage.js      — сохранение прогресса — НЕ редактировать
  js/app.js          — связывание — НЕ редактировать
```

Чтобы добавить новый корень-уровень — дописать **один объект** в массив `LEVELS`
в файле `js/data.js`. Больше ничего трогать не нужно.

## Как добавить новый уровень

Откройте `js/data.js` и добавьте объект в массив `LEVELS`:

```js
{
  id: 'mater',                    // уникальный код (латиницей, без пробелов)
  root: 'МАТЕР',                  // корень уровня
  note: "Один корень <span class='hl'>МАТЕР</span>. Краткое объяснение...",
  task: 'Что нужно сделать игроку в одном предложении.',
  hint: 'Подсказка — на чём держится опора (открытие списывает очки).',
  words: [
    // related: true  — слово ХРАНИТ корень (его надо ОСТАВИТЬ ✓)
    // related: false — омоним-самозванец по созвучию (его надо УБРАТЬ ✕)
    { lang: 'rus', w: 'МАТЬ',   tr: '',        mean: 'рус. — мать',            related: true },
    { lang: 'lat', w: 'MATER',  tr: 'матер',   mean: 'лат. — мать',            related: true },
    { lang: 'eng', w: 'MATTER', tr: 'мэтэр',   mean: 'англ. — вещество (другой корень)', related: false },
    { lang: 'lat', w: 'MATRIX', tr: 'матрикс', mean: 'лат. — матка, основа',   related: true, star: true }, // ★ бонус
  ],
}
```

Поля слова:
- `lang` — язык: `rus`, `bur`, `grc`, `lat`, `tur`, `eng` (новые языки — в карте `LANGS` выше).
- `w` — как слово пишется (можно в родном алфавите: ΤΥΡΟΣ, TÜRK).
- `tr` — транскрипция русскими буквами (можно оставить пустой `''`).
- `mean` — значение и перевод на русский.
- `related` — `true`, если слово хранит корень; `false`, если это омоним.
- `star` — `true` для задания со звёздочкой (даёт бонусные очки).

После сохранения файла просто обновите страницу — уровень появится сам.

## Очки и звания (настраиваются в `data.js`)

- `SCORE` — очки за слово, бонус за звёздочку, штраф за подсказку.
- `RANKS` — пороги званий.
- `DICT_URL` — какой словарь открывает кнопка «найти в словаре» (`{q}` = слово).

## Заготовки

В конце `LEVELS` лежат два пустых уровня-заготовки (`words: []`). Пока они пустые —
в игре не показываются. Заполните `root`, `note`, `task`, `hint` и `words` — и уровень
включится в прохождение автоматически.
